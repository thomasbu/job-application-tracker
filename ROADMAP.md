# Roadmap

Features and improvements planned for future iterations.

## In Progress

- Interview Prep UI — backend complete, frontend components are empty shells
  - Flashcards: flip-card review mode, confidence tracking
  - Coding Challenges: status board (TODO → In Progress → Completed)
  - Technical Questions: categorized Q&A list
  - Skills: proficiency level display
  - Study Sessions: session log with total time tracking

## Planned

### Features
- Dashboard with charts (applications per month, response rate, status breakdown)
- Kanban board view for applications (drag & drop between statuses)
- CSV export of applications
- Email reminders for upcoming interviews
- Mobile-responsive layout

### Technical
- GitHub Actions CI — automated test pipeline on every push
- Docker + docker-compose — containerized local setup
- Rate limiting on auth endpoints (brute force protection)
